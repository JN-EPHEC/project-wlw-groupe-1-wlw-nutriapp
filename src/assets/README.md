# Assets Replacement Instructions

## Temporary Placeholders

- `temp-logo.png` - Replace with your actual logo
- `temp-illustration.png` - Replace with your food bag illustration from Figma

## How to Replace

1. Export your assets from Figma as PNG files
2. Replace the temporary files in this directory
3. Keep the same filenames: `temp-logo.png` and `temp-illustration.png`
4. Restart your Expo development server
